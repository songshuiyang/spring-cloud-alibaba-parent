# spring-cloud-alibaba-parent

spring-cloud-alibaba 练习项目

### 模块介绍

|    子模块      |       描述      |    
|:----------|:------------- |
|   cloud-alibaba-config   |   使用nacos作为配置中心    | 


### 使用技术

* Spring Boot 1.5.7.RELEASE

* Spring Cloud Edgware.SR2
    * spring-cloud-starter-feign
* com.alibaba.cloud
    * spring-cloud-starter-alibaba-nacos-config （配置中心）
    * spring-cloud-starter-alibaba-nacos-discovery（服务注册）
